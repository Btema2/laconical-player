# Laconical Player

A modern, FOSS, local-first music player for Android built with Jetpack Compose and Media3. Inspired by Namida aesthetics.

## Features
- **Modern UI:** Built entirely with Jetpack Compose.
- **Media3 Integration:** High-performance audio playback using ExoPlayer.
- **Local-first:** Efficient database management with Room.
- **Modern Build System:** Powered by AGP 9.1 and Gradle 9.1 with built-in Kotlin support.

## Tech Stack
- **Language:** Kotlin 2.2
- **UI:** Jetpack Compose (Material 3)
- **Dependency Injection:** Hilt
- **Database:** Room
- **Media Playback:** Media3 (ExoPlayer)
- **Build System:** AGP 9.0, Gradle 9.1

## Project Status
**Pre-alpha:** Infrastructure and foundational setup are complete. Core playback and UI modules are initialized.

## How to Build
To build the debug APK, run:
```bash
./gradlew assembleDebug
```
